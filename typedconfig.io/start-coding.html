Tyco typed configuration

Start Coding with Tyco (Python)

This page contains quick instructions to install and use the Tyco Python bindings. Adjust commands for your environment as necessary.

Install

python -m pip install typedconfig-tyco

Basic usage

from tyco import load

with open('config.tyco') as f:
    cfg = load(f)

print(cfg['Application']['service'])

Run the CLI

# validate a file against the spec
> tyco validate config.tyco

# dump a parsed JSON representation
> tyco dump config.tyco --format json
