# Tyco (typed configuration) — Static site

This repository contains a minimal static, text-focused website for the Tyco configuration language. It uses only HTML5 and CSS3 and is responsive/mobile-first.

Local preview
1. Open `index.html` in a browser (double-click or serve it with a simple static server).

Quick static server (Python 3):
```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

What’s included
- `index.html` — main single-page site content
- `styles.css` — minimalist responsive styles
